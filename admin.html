⚡ Setup Guide — Supabase Integration

Step 1: Create a free project at supabase.com

Step 2: In Supabase, run this SQL to create the leads table:

create table leads (
  id uuid default gen_random_uuid() primary key,
  name text,
  mobile text,
  semester integer,
  subject text,
  note_title text,
  created_at timestamptz default now()
);

-- Allow public inserts
create policy "Allow public inserts" on leads
  for insert to anon with check (true);

-- Allow reads for authenticated
create policy "Allow reads" on leads
  for select to authenticated using (true);

alter table leads enable row level security;

Step 3: Copy your Project URL and anon key from Settings → API

Step 4: Open js/notes.js and js/admin.js and replace:

const SUPABASE_URL = 'YOUR_SUPABASE_URL';
const SUPABASE_KEY = 'YOUR_SUPABASE_ANON_KEY';
